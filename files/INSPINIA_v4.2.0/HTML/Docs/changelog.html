4.0.0 - 21 May 2025

A complete revamp to power your next-level admin experience!

Version 4.0 introduces a full redesign of INSPINIA, featuring more flexibility, modern UI standards, and powerful customization options.
  • Support for 6 Theme Skins – Easily switch between Classic, Material, Modern, SaaS, Flat, and Minimal skins using data-skin attributes.
  • Multiple Sidebar Variants – Includes Compact, Hover, Gradient, Offcanvas, Icon View, and more to fit any navigation need.
  • Topbar Options – Choose from Light, Dark, Gray, and Gradient topbars with enhanced layout control.
  • Expanded Layout Modes – Boxed, Scrollable, Horizontal, and other flexible layout structures to suit your application style.
  • Fully Modular Code – Built with SCSS, Gulp, and HTML partials for clean and scalable development.
